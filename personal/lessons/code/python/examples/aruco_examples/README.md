# 📷 Калибровка камеры квадрокоптера

Этот алгоритм позволяет откалибровать камеру квадрокоптера перед работой с ArUco-метками. Калибровка нужна, чтобы программа понимала, как правильно «считать» изображение: где находится объект в реальности и где он попал на фото.

## 🔍 Что такое калибровка камеры?

Калибровка — это процесс «настройки» камеры. После калибровки программа будет точно знать:
- как искажает изображение объектив,
- как пересчитать точку с фото в координаты реального мира.

Это необходимо, чтобы работать с ArUco-метками и другими системами компьютерного зрения.

---

## 🛠 Как выполнить калибровку

### 1. Подготовка

📄 Скачайте и распечатайте изображение шахматной доски:  
[Скачать шаблон](https://raw.githubusercontent.com/opencv/opencv/master/doc/pattern.png)

> ⚠️ Важно: печатайте **без масштабирования**, в **А4 формате**. Доска должна быть **чёткой и контрастной**.

---

### 2. Запуск программы

1. Подключите дрон и запустите скрипт `../advanced_navigation/19_map_creation_example1_calibration.py`.
2. Откроется окно с **видеопотоком с камеры дрона**.

---

### 3. Съёмка изображений

- Нажимайте клавишу **`1`**, чтобы сделать фото.  
  Сделайте **10–15 фотографий** с разных углов и расстояний.
- Когда закончите — нажмите **`Q`**, чтобы выйти из режима съёмки.

---

### 4. Проверка изображений

На экране по очереди появятся фотографии с обнаруженной шахматной сеткой.

- Нажимайте **любую клавишу**, чтобы переходить к следующему изображению.

---

### 5. Сохранение результата

После обработки программа сохранит файл с результатами калибровки:  
📁 **`data.yml`**

В этом файле хранятся все необходимые параметры камеры, и он будет использоваться в других программах (например, при работе с ArUco-метками).

> Убедитесь, что в папке с проектом появился файл `data.yml`. Без него работа с ArUco-метками будет невозможна.

---

## 🧠 Как работает алгоритм калибровки

```
camera = Camera()
images = get_images_from_drone_camera(camera)
mtx, dist = calibrate(images)
save_coefficients(mtx, dist, "data.yml")
```

- Создается объект camera класса Camera
- Функция get_images_from_drone_camera возвращает массив изображений с камеры дрона, который записывается в переменную images
- С помощью функции calibrate Рассчитываются коэффициенты калибровки (`mtx`, `dist`) — они описывают параметры объектива.
- Параметры сохраняются в `data.yml` с помощью функции save_coefficients.
