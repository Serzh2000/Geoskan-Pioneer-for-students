СИМУЛЯТОР PIONEER

ap.push(event)
Добавляет команду в очередь автопилота. Используется для взлета, посадки и других действий.
Аргументы: event (Ev.MCE_TAKEOFF, Ev.MCE_LANDING и др.)
ap.goToPoint(lat, lon, alt)
Полет в глобальные координаты (GPS). Координаты передаются как градусы * 10^7.
Аргументы: lat, lon (целые числа), alt (высота в метрах)
ap.goToLocalPoint(x, y, z, [time])
Полет в точку в локальной системе координат (в метрах). Можно указать время полета.
Аргументы: x, y, z (числа), time (секунды, опционально)
ap.updateYaw(angle)
Установка угла рыскания (курса) в радианах.
Аргументы: angle (число в радианах)
Timer.callLater(delay, func)
Выполняет функцию через указанное время (в секундах).
Аргументы: delay (секунды), func (функция)
Ledbar.new(count)
Создает объект для управления светодиодной лентой. У Pioneer обычно 4 светодиода.
Аргументы: count (число)
leds:set(i, r, g, b)
Устанавливает цвет i-го светодиода (0..N-1).
Аргументы: i (индекс), r, g, b (0.0 - 1.0)
camera.requestMakeShot()
Запрос на создание снимка камерой.
Gpio.new(port, pin, mode)
Создание объекта GPIO для управления пинами.
Аргументы: port (Gpio.A..E), pin (число), mode (Gpio.INPUT/OUTPUT)
Uart.new(num, rate)
Инициализация UART интерфейса.
Аргументы: num (номер), rate (скорость бод)
Spi.new(num, rate)
Инициализация SPI интерфейса.
Аргументы: num (номер), rate (скорость)
Ledbar.fromHSV(h, s, v)
Преобразование цвета HSV в RGB.
Аргументы: h (0-360), s (0-100), v (0-100)
