# Geoskan Pioneer — структура репозитория

## Структура верхнего уровня
- /personal — личные разработки автора курса
  - /lesson1 — материалы Урока 1 (code, data, tex)
- /geoskan — официальные материалы компании «Геоскан»
  - /Программирование/Уроки — уроки 2+ и прочие официальные разделы
  - FAQ, официальные PDF и методички
- /vendor — внешняя документация и оригинальные материалы
  - /pioneer-api-docs — локальная HTML-документация API Pioneer
  - /fs-i6s-docs — руководства по пульту FlySky FS-i6S
- Файлы конфигурации: .gitignore, .gitattributes, README.md

## Авторство и условия использования
- Содержимое каталога /geoskan принадлежит компании «Геоскан»; распространяется на условиях, указанных в соответствующих документах компании.
- Содержимое каталога /personal принадлежит автору курса; разрешено использование в образовательных целях, кроме случаев, отдельно оговорённых автором.
- При публикации материалов указывайте первоисточник и автора.

## Навигация
- Личные материалы: /personal/lessons/{code,data,tex}
- Официальные материалы: /geoskan/{Программирование/Уроки, FAQ, *.pdf}

## Лицензии и права
- Лицензионные файлы и условия использования размещаются в соответствующих директориях /geoskan и /personal.
- В случае отсутствия явного файла лицензии — применяется политика компании/автора, указанная в сопроводительных документах.

## Исключения из Git
- Временные и системные файлы исключаются через .gitignore (логи, кэши сборки, артефакты LaTeX, системные файлы ОС).
