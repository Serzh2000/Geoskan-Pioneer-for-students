## 📖 Учебные материалы и инструменты

В этом репозитории собраны все необходимые материалы для успешного освоения программирования БПЛА. Основной упор делается на образовательную платформу **Geoskan Pioneer**.

### 📚 Учебник
Полное методическое пособие, содержащее теоретическую базу и практические задания.
* 📄 **PDF-версия:** [Открыть учебник](./personal/lessons/tex/main.pdf) *(если скомпилирована)*
* 🖋️ **Исходники LaTeX:** [Перейти к исходникам](./personal/lessons/tex/main.tex)

### 🎮 Симулятор дрона
Специально разработанный веб-симулятор позволяет безопасно тестировать код (Lua/Python) и отлаживать автономные миссии без риска повредить реальное оборудование.
* 🌐 **Запустить симулятор:** [simulator.sakoryagin.ru](https://simulator.sakoryagin.ru)
* 💻 **Исходный код симулятора:** Находится в папке [`/personal/lessons/code/lua/web-simulator`](./personal/lessons/code/lua/web-simulator)

## 📂 Структура репозитория

Для удобства навигации все файлы строго структурированы по директориям:

### 👨‍💻 1. Личные разработки (`/personal`)
Оригинальные материалы и код, созданные автором курса:
* 📁 **`/personal/lessons/tex`** — исходники учебника в формате LaTeX.
* 📁 **`/personal/lessons/code`** — практические примеры и задания:
  * 🔹 **`.../lua/`** — скрипты для автопилота на языке Lua, включая симуляторы.
  * 🔹 **`.../python/`** — скрипты машинного зрения и продвинутой навигации на Python.
* 📁 **`/personal/lessons/data`** — дополнительные датасеты и ресурсы.

### 🏢 2. Официальные материалы (`/geoskan`)
Материалы, предоставленные производителем оборудования:
* 📁 **`/geoskan/Программирование/Уроки`** — официальные методические указания от компании.
* 📁 **`/geoskan/FAQ`** — ответы на частые вопросы, схемы сборки и официальные PDF-руководства.

### 📚 3. Внешняя документация (`/vendor`)
Справочная информация от сторонних разработчиков:
* 📁 **`/vendor/pioneer-api-docs`** — локальная HTML-копия документации API Geoskan Pioneer.
* 📁 **`/vendor/fs-i6s-docs`** — инструкции по настройке и калибровке пульта FlySky FS-i6S.
